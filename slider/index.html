Name of Invention: HTML

Developed By: Tim berners-lee

year: 1989 - 1991

Bio: Sir Tim Berners-Lee, born in 1955 in London, is a British computer scientist who invented the World Wide Web in 1989 while working at CERN. He created HTML, the first web browser, and key web technologies like URLs and HTTP, laying the foundation of the modern internet.

Purpose: was created to structure and link documents on the web, making it possible to share information globally.
